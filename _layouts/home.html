fabric management done right

Quattor is a system administration toolkit providing a powerful, portable, and modular set of tools for the automated installation, configuration, and management of clusters and farms.

It is developed as a community effort and provided as open-source software (SourceForge, GitHub) under the Apache 2 license and EU DataGrid Software License.

Complete

Quattor is a set of tools that work well together to manage the complete lifecycle of machines—from initial installation, through configuration, to service upgrades.

Flexible

The tools interact via well defined interfaces, allowing alternative implementations and integration existing site-specific tools.

Scalable

Used in academic and commercial environments, managing infrastructures ranging from tens of nodes to over 20000.

Accurate

Extensive validation features for machine descriptions catch configuration errors before they are deployed onto your infrastructure.
